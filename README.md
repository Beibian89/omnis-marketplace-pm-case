# Omnis — сервис доставки рецептурных препаратов

Продуктовый кейс: Разработка нового направления на базе существующего сервиса.

## О проекте

Omnis — маркетплейс безрецептурных препаратов. Задача — разработать сервис доставки рецептурных препаратов на базе этого маркетплейса: от идеи до roadmap и плана масштабирования.

## Что сделано

- Разбор целевой аудитории и Job Story
- Анализ рынка и конкурентов
- Описание продукта
- Продуктовый роадмап
- Прототипы
- Выбор модели монетизации
- Прогноз GMV
- Риск-анализ
- Описание шагов для масштабирования
